result_queue ^rq-title
• asyncio.Queue (async)
• Fed by: loop.call_soon_threadsafe()

• Drained by: _drain_tool_results() ^rq-body

Layer 3: stream_generator() — streaming_routes.py ^layer3-label

stream_generator() ^sg-title
• Consumes stream_async() via _sdk_with_keepalive() wrapper
• Drains result_queue between events
• Keepalive: `: keepalive\n\n` every 20s
• Filters: empty-name tool_result
• Error: exception → error SSE event
• Fallback: no text → "[No response generated]"
• writer.write_text / write_tool_use / write_tool_result / write_complete / write_error ^sg-body
