## Architecture Truth

OpenClaw is built ON TOP of Pi. Confirmed via live instance package.json:

```
@mariozechner/pi-agent-core      0.53.0  → agent types, tool definitions
@mariozechner/pi-ai              0.53.0  → LLM calls, streaming, OAuth
@mariozechner/pi-coding-agent    0.53.0  → tools (read/write/edit/bash), SKILL.md loading, estimateTokens
@mariozechner/pi-tui             0.53.0  → terminal UI
```

```
Pi (Mario Zechner)
├── LLM calls / streaming / OAuth
├── Core tools: read, write, edit, bash
├── SKILL.md loading (loadSkillsFromDir)
└── estimateTokens, NormalizedUsage

OpenClaw wraps Pi and adds:
├── Gateway daemon (keeps Pi alive 24/7)
├── Plugin SDK (TypeScript API over Pi internals)
├── Multi-channel messaging (Telegram, Discord, WhatsApp, Slack, Line, Lark...)
├── Cron scheduler
├── Web dashboard + iOS/macOS apps
└── Plugin marketplace
```

**Key distinction:** Dan's video showed Pi's **TUI extension system** (`pi.on("tool_call", ...)`).
That's the terminal layer. OpenClaw has its own **Plugin SDK** one layer above.
His code can't be copy-pasted but every pattern is achievable via the Plugin SDK.

**Plugin SDK API** (different from Pi TUI API):
```typescript
// Pi TUI (Dan's extensions — NOT for OpenClaw):
pi.on("tool_call", handler)
pi.registerTool(...)
pi.registerCommand(...)

// OpenClaw Plugin SDK (what we use):
api.registerTool({ name, description, parameters, execute })
api.onReply(callback)
api.onToolCall(callback)
```

## Plugin Structure (OpenClaw)

```
~/.openclaw/plugins/<plugin-id>/
├── openclaw.plugin.json     ← REQUIRED: manifest + JSON Schema
├── package.json             ← "openclaw": { "extensions": ["./index.ts"] }
└── index.ts                 ← plugin entry point
```

```json
// openclaw.plugin.json (minimum required)
{
  "id": "token-counter",
  "name": "Token Counter",
  "description": "Appends token count and cost to every reply",
  "configSchema": {
    "type": "object",
    "additionalProperties": false,
    "properties": {}
  }
}
```

Enable in `~/.openclaw/openclaw.json`:
```json
{
  "plugins": {
    "entries": {
      "token-counter": { "enabled": true }
    }
  }
}
```

## Phase 1 — Token Counter Plugin (2 hrs, if Phase 0 config doesn't exist)

Build a custom plugin that uses OpenClaw's existing `formatResponseUsageLine`:

```
~/.openclaw/plugins/token-counter/
├── openclaw.plugin.json
├── package.json
└── index.ts
```

```typescript
// index.ts
import { formatTokenCount, formatUsd, estimateUsageCost, resolveModelCostConfig } from "openclaw/plugin-sdk";

export default function(api) {
  api.onReply(async ({ usage, model, provider, config }, append) => {
    const costConfig = resolveModelCostConfig({ provider, model, config });
    const cost = estimateUsageCost({ usage, cost: costConfig });
    const line = [
      `💬 ${formatTokenCount(usage?.input)} in`,
      `${formatTokenCount(usage?.output)} out`,
      usage?.cacheRead ? `📦 ${formatTokenCount(usage.cacheRead)} cache` : null,
      cost ? `$${formatUsd(cost)}` : null,
    ].filter(Boolean).join(" · ");
    append(`\n\`${line}\``);
  });
}
```

## Phase 2 — Damage Control Plugin (3 hrs)

Port Dan's `.pi/damage-control-rules.yaml` to an OpenClaw plugin.

**Rules file** (same YAML format as Dan's):
```yaml
# ~/.openclaw/plugins/damage-control/damage-control-rules.yaml
blocked_bash_patterns:
  - "rm -rf"
  - "rm -f"
  - "git reset --hard"
  - "git push --force"
  - "DROP TABLE"
  - "DROP DATABASE"
  - "TRUNCATE TABLE"
  - "aws.*delete"
  - "gcloud.*delete"

zero_access_paths:
  - "**/.env*"
  - "**/.ssh/**"
  - "**/id_rsa*"
  - "**/credentials.json"
  - "**/*.pem"
  - "**/kubeconfig*"

read_only_paths:
  - "/etc/**"
  - "~/.bashrc"
  - "~/.bash_profile"
```

```typescript
// index.ts
export default function(api) {
  api.onToolCall(async ({ name, params }, { block, warn }) => {
    if (name === "bash") {
      const matched = findBlockedPattern(params.cmd, rules.blocked_bash_patterns);
      if (matched) return block(`🛡️ Damage Control: blocked \`${matched}\``);
    }
    if (name === "read" || name === "write" || name === "edit") {
      if (isZeroAccess(params.path, rules.zero_access_paths)) {
        return block(`🛡️ Damage Control: zero-access path ${params.path}`);
      }
    }
  });
}
```
